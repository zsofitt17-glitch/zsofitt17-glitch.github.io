Heatmaps Based on Postal Code Data

The following heatmaps compare postal code level indicators using two different classification methods. The image on the left shows the equal interval version, while the image on the right shows the quantile version.

Equal interval classification divides the full data range into equally sized value intervals. Quantile classification places approximately the same number of postal code areas into each class.
