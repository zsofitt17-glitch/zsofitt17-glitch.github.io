Flow Visualization Using a Web-Based Tool

The following visualization presents movement patterns between bike-sharing stations using a flow map created in Flowmap.blue. The visualization is based on origin-destination data from May 2023, where the connections between locations represent the direction and intensity of trips.

Only departure trips from May 2023 were included, as this already provided a sufficiently large dataset for identifying the main movement patterns within the network. Flowmap.blue is a ready-to-use web tool that allows flow maps to be created without custom programming, making it accessible even for users with limited coding experience.
